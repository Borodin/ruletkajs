RuletkaJS
=========
[![Build Status](https://travis-ci.org/Borodin/ruletkajs.svg?branch=master)](https://travis-ci.org/Borodin/ruletkajs)
#### [ruletkajs.com](http://www.ruletkajs.com/)

#### Getting started
```npm install -g gulp bower```

```npm install ```

#### Build
```gulp ```

#### Run Server
```node server ```

#### Run BrowserSync
```gulp serve ```
